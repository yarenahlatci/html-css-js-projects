## NASIL KULLANILIR?

1. Projeyi klonlayın:
    ```sh
    git clone https://github.com/kullanici-adi/web-projects.git
    ```
2. İlgili proje klasörüne gidin:
    ```sh
    cd PROJECT_NAME
    ```
3. Projeyi tarayıcıda görüntülemek için HTML dosyasını açın.
